Enter the unique Item ID for the secret in your Bitwarden vault.

Using the ID is the most reliable way to look up an item, as it is always unique. You can find the Item ID by opening the item in the Bitwarden Web Vault and copying it from the browser's address bar.

The ID is the long string of characters that comes immediately after itemId= in the URL.

For example, in the URL https://.../?itemId=a1b2c3d4-e5f6-7890-a1b2-c3d4e5f67890&action=view, the ID is a1b2c3d4-e5f6-7890-a1b2-c3d4e5f67890.
